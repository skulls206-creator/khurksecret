What is Shamir's Secret Sharing?

Shamir's Secret Sharing (SSS) splits a secret into N shares, where any K of those shares (the threshold) can reconstruct the original secret. Fewer than K shares reveal nothing about the secret. This implementation runs entirely in your browser — no data is ever sent to a server.

⚠ Security: Distribute shares to different people. Never store all shares together. This is a browser-based demo — for real secrets, use an audited library or hardware security module.
